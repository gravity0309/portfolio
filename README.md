# Portfolio Website

A modern, responsive developer portfolio built with React, TypeScript, and Tailwind CSS.

## Overview

This portfolio showcases my projects, technical skills, certifications, achievements, and educational background in Computer Science and Artificial Intelligence.

## Features

* Responsive design
* Minimal black-and-white theme
* Project showcase section
* Skills and technology stack
* Certifications and achievements
* Resume download
* Contact section
* Smooth navigation

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS

## Featured Projects

### DocuMind

AI-powered PDF Question Answering system built using React, FastAPI, LangChain, FAISS, Sentence Transformers, and Groq API.

### Darshan Ease

Temple visitor management platform built with the MERN stack.

### Complaint Management System

Complaint tracking platform with JWT authentication and role-based access control.

### Diabetes Prediction System

Machine learning project using Random Forest classification for healthcare prediction.
